# [collision.html](https://dudushy.github.io/collision.html/)
A simple collision game (attempt).

---
## Prerequisites:
- [NodeJS](https://nodejs.org/)

## How to install:
1. Install packages
    ```bash
    npm i
    ```
2. Run project
    ```bash
    npm start
    ```
3. Build project
    ```bash
    npm run build
    ```
